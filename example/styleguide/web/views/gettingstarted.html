New to AngularDart?

Before getting into wsk_angular, it might be helpful to read about the framework. You might also be interested to read a bit about the Material Design specs.

wsk_material is using Google's Web Starter Kit(WSK) as base CSS framework. At the moment WSKs version is 5.x but wsk_material is using the upcoming 6.x version

Web Starter Kit vs. Bootstrap

You all know these arguments - Bootstrap is heavy!, you have no SASS-support and so forth.
Google’s Web Starter Kit is a little bit different. Instead of providing you with every CSS element that you could ever use, the WSK takes an exclusionary approach, providing you with a fairly simple base on which to build. If you are familiar with Foundation, what you get CSS-wise is similar to their ‘Essentials’
If you want to read further - I found a good comparison on this page

How do I start?
  1. Fork one of these samples
  2. Update your pubspec.yaml 
    name: wsk_angular_layout_header_drawer_sample
version: 0.0.1
description: An absolute bare-bones web app.

environment:
    sdk: '>=1.0.0 <2.0.0'

dependencies:
    browser: any

    angular: ">=1.0.0 <2.0.0"

    # Change this to ANY
    wsk_angular: any

dev_dependencies:
    unittest: any

#------------------------------------------------
transformers:
    - angular:

    # List is necessary because of this bug: http://goo.gl/4Rb3wr
    html_files: [
        'web/index.html'
    ]
  3. pub update
  4. pub build
