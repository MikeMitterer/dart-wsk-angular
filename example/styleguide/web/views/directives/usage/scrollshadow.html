Basic Usage Html CSS Place the attribute "scrollshadow" on the element that scrolls. Add the class wsk-add-scrollshadow-here to the element that should have the shadow. Usually this is wsk-layout-header. The "element that scrolls" is in most cases either wsk-layout-content... 
<wsk-layout header="fixed">
    <wsk-layout-header class="wsk-add-scrollshadow-here">
        ...
    </wsk-layout-header>
    <wsk-drawer>
        ...
    </wsk-drawer>
    <wsk-layout-content scrollshadow>
        <section>
            ...
        </section>
        <footer class="wsk-mini-footer-flex">
            ...
        </footer>
    </wsk-layout-content>
</wsk-layout>
... but can also be the first element within wsk-layout-content - like it is in the layout-header-drawer-footer sample. The "layout-header-drawer-footer" sample uses a flexbox based layout with fixed header + fixed footer 
<wsk-layout header="fixed">
    <wsk-layout-header class="wsk-add-scrollshadow-here">
        ...
    </wsk-layout-header>
    <wsk-drawer>
        ...
    </wsk-drawer>
    <wsk-layout-content >
        <!-- scrollshadow - section ist set to overflow: scroll -->
        <section scrollshadow>
            ...
        </section>
        <footer class="wsk-mini-footer-flex">
            ...
        </footer>
    </wsk-layout-content>
    <footer class="wsk-bottom-bar">
        ...
    </footer>
</wsk-layout>

Nothing to do here. wsk-shadow--z2 is automatically added for you.
